## Инструкции по сборке и запуску проекта
Запустить программу можно с помощью комманды:
    ```
	psql -f ./bag.sql
    java -jar target\pr4.jar
    ```

Собрать программу с нуля можно с помощью выполнения последовательно комманд:
    ```
    mvn clean
    mvn compile
    mvn package
    mvn install
    ```
